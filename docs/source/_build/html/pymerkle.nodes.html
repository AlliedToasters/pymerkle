pymerkle.nodes module¶

Provides the base class for the Merkle-tree’s nodes and an inheriting class for its leaves

class pymerkle.nodes.node(hash_function, encoding, record=None, left=None, right=None)[source]¶

Bases: object

Base class for the nodes of a Merkle-tree

Parameters:
  • record (str or bytes or bytearray) – [optional] the record to be encrypted within the node. If provided, then the node is considered to be a leaf
  • left (nodes.node) – [optional] the node’s left parent. If not provided, then the node is considered to be a leaf
  • right (nodes.node) – [optional] the node’s right parent. If not provided, then the node is considered to be a leaf
  • hash_function (method) – hash function to be used for encryption. Should be the .hash method of the containing Merkle-tree
  • encoding (str) – Encoding type to be used when decoding the hash stored by the node. Should coincide with the containing Merkle-tree’s encoding type.
Variables:
  • stored_hash – (bytes) The hash currently stored by the node
  • left – (nodes.node) The node’s left parent. Defaults to None if the node is a leaf
  • right – (nodes.node) The node’s right parent. Defaults to None if the node is a leaf
  • child – (nodes.node) The node’s child parent. Defaults to None if the node is a root
  • hash_function – (method) The hash function used by the node when recalcullating hashes. For interior nodes it should coincide with the .hash attribute of the containing Merkle-tree. For leaf nodes it is None, since their hash is never recalculated.
  • encoding – (str) Used by the node for decoding its hash. Should coincide with the encoding type of the containing Merkle-tree.

descendant(degree)[source]¶

Detects and returns the node that is degree steps upwards within the containing Merkle-tree

Note

Descendant of degree 0 is the node itself, descendant of degree 1 is the node’s child, etc.

Parameters:degree (int) – depth of descendancy. Must be non-negative
Returns:the descendant corresdponding to the requested depth
Return type:nodes.node

Note

Returns None if the requested depth of dependancy exceeds possibilities

recalculate_hash()[source]¶

Recalculates the node’s hash under account of its parents’ new hashes

This method is to be invoked for all non-leaf nodes of the Merkle-tree’s rightmost branch every time a new leaf is appended into the tree

Warning

Only for interior nodes (i.e., with two parents); fails in case of leaf nodes
